📝 Step 1: Create Your GA4 Property

Set up your analytics account from scratch

If you don't have a GA4 property yet, here's how to set it up. If you already have one, skip to Step 2.

📋 Step‑by‑Step
  1. Go to analytics.google.com
  2. Click "Admin" (gear icon)
  3. In the "Property" column, click "Create Property"
  4. Enter your business name and website URL
  5. Select your industry category and time zone
  6. Click "Next" and select your business objectives
  7. Click "Create" – your GA4 property is ready
🇰🇪 Kenya‑Specific Settings
  • Time zone: Select "Africa/Nairobi" (EAT)
  • Currency: Select "KES" (Kenya Shillings)
  • Industry: Choose the category closest to your business
  • Data sharing: Keep default settings

💡 Kenya pro tip: If you have multiple websites or brands, create separate GA4 properties for each. This keeps your data clean and prevents confusion. Use clear naming conventions like "Business Name – Website" and "Business Name – App."

2

🔧 Step 2: Install Tracking Code

Add GA4 to your website

Now it's time to install the tracking code on your website. There are multiple ways to do this – choose the method that works best for you.

📋 Installation Methods
  • Google Tag Manager (GTM): Recommended – easiest to manage
  • Direct code: Add to <head> of your website
  • Shopify: Add to theme.liquid or use Shopify's GA4 integration
  • WordPress: Use plugin (Site Kit, MonsterInsights, etc.)
  • Wix/Squarespace: Built‑in integration
🔧 How to Get Your Tracking ID
  1. In GA4, click "Admin" → "Data Streams"
  2. Click "Add Stream" → "Web"
  3. Enter your website URL and stream name
  4. Click "Create Stream"
  5. Copy the measurement ID (starts with G‑XXXXXXXX)

💡 Kenya pro tip: We recommend Google Tag Manager for most businesses. It gives you centralized control over all tracking codes (GA4, Facebook Pixel, LinkedIn, etc.) and allows you to add or update tracking without touching your website code – perfect for Kenyan businesses where technical resources may be limited.
